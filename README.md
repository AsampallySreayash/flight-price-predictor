# ✈️ Flight Price Prediction System

A Machine Learning-based web application that predicts flight ticket prices using Random Forest.

## 🚀 Features
- Predict flight ticket prices
- User login & registration system
- Search history tracking
- Interactive UI

## 🛠️ Tech Stack
- Python, Flask
- Machine Learning (Random Forest)
- HTML, CSS, JavaScript
- SQLite

## ⚙️ Working
1. User enters flight details
2. Data goes to Flask backend
3. Data is encoded
4. Model predicts price
5. Result is shown

## ▶️ Run Project
pip install flask pandas numpy scikit-learn  
python app.py
